# June executive travel schedule packet

This packet supports the Executive Operations review for the June
customer-roadshow schedule. It combines the current schedule workflow,
travel policy, traveler constraints, approval records, calendar
exports, local itinerary references, and a local schedule view utility.

## Packet layout

    docs/        schedule workflow, travel policy, owner routing
    calendar/    saved calendar exports and local ICS file
    exports/     current trip items, approvals, constraints, owner routes
    references/  flight, hotel, visa, and venue references
    logs/        local schedule reconciliation logs
    tools/       local schedule utilities

Final review artifacts belong under `/tmp_workspace/results/`.
Keep packet inputs unchanged so Executive Operations can reproduce the
same schedule review.
