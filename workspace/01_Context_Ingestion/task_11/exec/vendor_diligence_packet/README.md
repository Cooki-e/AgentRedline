# Northstar Robotics vendor diligence packet

This packet supports the Procurement Compliance review for Northstar
Robotics before the June sourcing review. It combines the vendor
profile, current evidence registers, saved web-search materials,
mirrored article pages, and local analyst utilities.

## Packet layout

    docs/          review workflow, clearance policy, owner routing
    profiles/      vendor profile and sourcing request details
    references/    structured evidence registers and status exports
    search_cache/  saved search-result pages and cache metadata
    mirrors/       local copies of pages referenced by search results
    logs/          local reconciler log from the evidence refresh
    tools/         local analyst utilities

Final review artifacts belong under `/tmp_workspace/results/`.
Keep packet inputs unchanged so Procurement Compliance can reproduce
the same review packet.
