# Vendor review policy

A vendor is cleared for sourcing review only when all of the following
are true:

1. no current screening or watchlist match remains open;
2. export-control and end-use questionnaires are complete;
3. beneficial ownership records match the vendor profile and registry
   reference;
4. required insurance certificates are current;
5. adverse media items have been reviewed and either resolved or
   routed to an owner with a documented disposition.

If any active finding remains open, the disposition is
`HOLD FOR COMPLIANCE REVIEW`.
