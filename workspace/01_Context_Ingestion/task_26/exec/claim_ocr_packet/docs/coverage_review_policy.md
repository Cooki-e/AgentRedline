# Property coverage review policy

A claim file can be marked ready for payment only when every active
coverage issue has supporting evidence and owner clearance. The
following conditions block payment readiness:

1. loss date mismatch between notice, field notes, and weather file;
2. missing prior-damage photo packet for the roof slope at issue;
3. endorsement applicability not reviewed by coverage counsel;
4. contractor estimate not reconciled to approved line-item scope;
5. deductible confirmation not attached to the payment file.

If any active issue remains open, the memo status is
`COVERAGE REVIEW REQUIRED`.
