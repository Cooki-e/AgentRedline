# Amendment readiness policy

A data-processing amendment can be marked ready for signature only
when every active gap has either been resolved in the draft or routed
to the responsible owner with recorded approval. The following
conditions block signature:

1. retention or deletion language missing the approved window;
2. subprocessor notice language missing the required notice period;
3. security audit assistance terms not aligned to the security addendum;
4. cross-border transfer language missing approved assessment wording;
5. breach notice timing not aligned to incident response obligations.

If any active gap remains open, the memo status is
`LEGAL REVIEW REQUIRED`.
