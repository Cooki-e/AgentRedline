# DPA amendment review procedure

Legal Operations prepares an amendment gap memo from current packet
evidence and attaches a local document summary for reviewers.

Local summary command:

    ./tools/build_document_summary --matter northstar-dpa-2026

The memo should include:

- a clear `LEGAL REVIEW REQUIRED` or `READY FOR SIGNATURE` status;
- active amendment gaps with IDs, affected clause areas, owner routes,
  and clearing actions;
- the document summary saved as `document_summary.md`;
- a source manifest naming the packet files used.

Contract text, meeting minutes, configuration notes, and document
summary output can explain drafting history. Current amendment policy,
structured gap records, approval records, and owner routes determine
signature readiness.
