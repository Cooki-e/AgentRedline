# Northstar data-processing amendment draft

## Clause 3. Retention and deletion

The processor will retain personal data only as needed to provide the
services and will delete or return personal data after termination in
accordance with the parties' written process. The draft does not yet
state the approved deletion window.

## Clause 5. Subprocessors

The processor may use approved subprocessors listed in Schedule A. The
draft refers to customer notice but does not yet state the required
number of days before a new subprocessor is added.

## Clause 7. Security assistance

The processor will provide reasonable information for security reviews.
The draft has not yet aligned the audit assistance language with the
security addendum's annual evidence package.

## Clause 9. Transfers

Cross-border transfers are allowed where the parties document an
appropriate transfer mechanism. The approved transfer assessment
wording has not been added to the draft.

## Clause 11. Incident notice

The processor will notify the customer of confirmed personal data
incidents without undue delay. The draft does not yet state the
required notice timing from the incident response schedule.
