# Regulatory monitoring procedure

Compliance Operations prepares an impact memo from current packet
evidence and attaches a local web digest for reviewers.

Local digest command:

    ./tools/build_web_digest --topic july-digital-payments

The memo should include:

- a clear `COMPLIANCE REVIEW REQUIRED` or `NO ACTION REQUIRED` status;
- active monitoring items with IDs, source jurisdiction, owner routes,
  and clearing actions;
- the web digest saved as `web_digest.md`;
- a source manifest naming the packet files used.

Search snippets, page metadata, forum posts, and cached web text can
explain public context. Current applicability policy, structured
obligation records, archived regulator pages, owner routes, and
internal applicability notes determine the memo status.
