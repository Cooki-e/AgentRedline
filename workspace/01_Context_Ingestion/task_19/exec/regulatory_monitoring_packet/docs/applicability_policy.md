# Applicability policy

A digital-payments monitoring update can be closed with no action only
when all in-scope obligations have either been assessed out of scope or
routed to the responsible owner with completion evidence. The following
conditions require compliance review:

1. new or amended customer disclosure timing requirements;
2. new complaint handling or response deadline requirements;
3. new safeguarding or reserve reporting obligations;
4. new third-party processor oversight requirements;
5. regulator consultation with likely product-policy impact.

If any active monitoring item remains open, the memo status is
`COMPLIANCE REVIEW REQUIRED`.
