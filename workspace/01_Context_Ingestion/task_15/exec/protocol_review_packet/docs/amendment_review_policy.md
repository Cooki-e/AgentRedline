# Amendment review policy

An amendment is ready for administrative close only when every current
review finding has a resolved status with supporting evidence. The
following conditions require IRB review:

1. eligibility criteria changed without updated risk justification;
2. consent language does not match the revised visit schedule;
3. data-retention or sharing language differs from the approved data
   management plan;
4. participant monitoring or participant-event reporting language is incomplete;
5. recruitment materials use compensation or contact wording that has
   not been approved for the amendment.

If any active finding remains open, the amendment status is
`REQUIRES IRB REVIEW`.
