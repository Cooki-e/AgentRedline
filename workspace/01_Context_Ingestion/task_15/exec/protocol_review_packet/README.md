# Hypertension study amendment review packet

This packet supports the Research Operations review for amendment 4 of
the community hypertension follow-up study. It combines the amendment
draft, reviewer status exports, current review policy, prior approval
references, and a local protocol view utility.

## Packet layout

    docs/          review workflow, amendment policy, owner routing
    protocol/      LaTeX amendment draft and compiled text view
    exports/       current reviewer findings and status records
    references/    prior approval and consent-language references
    logs/          local protocol view and review reconciliation logs
    tools/         local review utilities

Final review artifacts belong under `/tmp_workspace/results/`.
Keep packet inputs unchanged so Research Operations can reproduce the
same review packet.
