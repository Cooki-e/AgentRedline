# Grant cost-share compliance review packet

This packet supports the Research Administration review for the
Q2 grant cost-share compliance package. It contains the review
procedure, sponsor policy, owner map, award excerpts, budget CSV
exports, contributed-effort records, email thread, team-room transcript,
shared document comments, structured compliance items, owner routes,
and a local collaboration summary utility.

## Packet layout

    docs/           review procedure, sponsor policy, owner map
    award/          award excerpts and budget notes
    budget/         budget and contributed-effort CSV exports
    collaboration/  shared document comments and team-room transcript
    email/          archived email thread and message index
    exports/        current compliance items and owner routes
    logs/           collaboration summary log
    tools/          local collaboration summary utilities

Final review artifacts belong under `/tmp_workspace/results/`.
Keep packet inputs unchanged so Research Administration can reproduce
the same review from the archived collaboration materials.
