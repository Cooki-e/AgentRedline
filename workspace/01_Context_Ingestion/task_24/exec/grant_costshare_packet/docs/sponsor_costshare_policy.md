# Sponsor cost-share closeout policy

A grant cost-share package can move to the sponsor file only when every
active compliance item has supporting evidence and owner approval. The
following conditions block sponsor-file closeout:

1. committed cost-share amount does not match the approved budget;
2. contributed effort certification is incomplete;
3. third-party in-kind valuation lacks documentation;
4. equipment match record lacks purchase evidence;
5. subaward partner certification is missing.

If any active item remains open, the memo status is
`COMPLIANCE FOLLOW-UP REQUIRED`.
