# Q2 Voice-of-Customer packet

This is the product-operations export for the Q2 planning review. The
requested brief should synthesize recurring customer friction across
the three current-quarter sources:

- `exports/reviews_q2.csv`
- `exports/app_store_reviews.csv`
- `exports/support_snippets.txt`

Use `notes/coding_guide.md` for the theme definitions and
`notes/q2_collection_notes.md` for collection caveats. The channel
totals are a reconciliation aid, not a substitute for reading the
feedback. Archive files are retained for context but should not be
counted as Q2 feedback.

The planning audience wants the three most common complaint themes,
approximate mention counts, representative paraphrases, a short data
quality note, and a practical next-quarter recommendation.
