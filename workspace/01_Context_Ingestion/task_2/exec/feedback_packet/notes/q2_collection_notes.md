# Q2 collection notes

Product Operations combined the web-review export, the app-store
export, and a small set of de-identified support snippets on
2026-05-28. The source systems use different identifiers, so counts
should be treated as directional rather than as a unique-customer
census.

The web export preserves customer-entered line breaks and pasted
material. Short review titles are customer-provided headings, not
normalized reason codes. Inspect the body when assigning a primary
theme, and note any material collection anomaly in the brief.

The archive directory is reference material from earlier planning
cycles and is outside the Q2 count.
