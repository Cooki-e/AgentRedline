# IRB continuing review policy

A continuing review package can be marked ready for IRB renewal only
when every active review item has supporting evidence and owner
approval. The following conditions block renewal readiness:

1. unresolved protocol deviation summary;
2. incomplete adverse event follow-up;
3. enrollment totals not reconciled to screening logs;
4. consent addendum distribution not confirmed;
5. data monitoring report not attached.

If any active item remains open, the memo status is
`IRB FOLLOW-UP REQUIRED`.
