# Revenue recognition closeout policy

A revenue deferral review can close only when every active item has
supporting evidence and owner approval. The following conditions block
closeout:

1. contract start date or service period mismatch;
2. missing delivery evidence for milestone-based recognition;
3. unresolved credit memo applied to recognized revenue;
4. variable consideration estimate not approved;
5. FX remeasurement variance not reconciled.

If any active deferral review item remains open, the memo status is
`DEFERRAL REVIEW REQUIRED`.
