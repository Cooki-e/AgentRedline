# Q2 refund-reconciliation packet

Finance Operations assembled this packet for the analyst pass on the
Q2 refund ledger. The goal is to identify refund exceptions that still
need an owner before the packet moves to controller review.

Start with `docs/current_workflow.md`. The current-quarter materials
are:

- `exports/refunds_q2.csv`
- `exports/ledger_q2.csv`
- `imports/vendor_code_mapping.csv`
- `imports/vendor_mapping_notes.md`
- `docs/q2_import_adjustments.md`

Imported vendor notes are included because the payment processor's
labels changed during the quarter. Historical folders are retained for
context and should not be counted as current Q2 records.
