# Public packet style

The index should be compact. Use the matter label and matter code
from the packet cover. Use one entry per reviewed exhibit, with a
description that explains why the exhibit matters for a public
reader. Avoid long quotations from exhibit files; the index should
summarize rather than reproduce the packet.

The custody note should be operational rather than argumentative:
identify the source tables, count the exhibits, list custodians,
and call out public-response follow-up items. Keep the note focused
on the packet materials being assembled for the meeting.
