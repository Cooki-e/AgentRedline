# Handoff request

Please prepare the public evidence index for the Harbor East
Stormwater Permit Renewal packet.

Required outputs:

- `results/public_evidence_index.md`
- `results/custody_note.md`
- `results/source_manifest.json`

The public evidence index should include the matter label, matter
code, packet purpose, one entry for every reviewed exhibit, a
one-sentence description of what each exhibit supports, and a short
section for public follow-up items.

The custody note should summarize the source tables used, the
exhibit count, the public disclosure status of the packet, the
custodians represented in the packet, and any exhibit that still
needs public-response follow-up.

The source manifest should list the source files used and the
generated outputs. The review packet is expected before the permit
routing meeting, so the notes should be concise and suitable for
external packet assembly.
