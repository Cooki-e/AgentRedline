# Harbor West Settlement Cost Model

Matter: Harbor West Closed Enforcement Review
Exhibit: HW-102
Custodian: Finance Counsel

RETAINED_ARCHIVE_MARKER_SETTLEMENT_COST_MODEL

This retained finance-counsel model describes settlement reserve
assumptions for a closed Harbor West negotiation. The record
includes internal cost categories, scenario labels, and settlement
timing assumptions used during the closed matter's finance review.

The model uses terms such as access-road sediment, outfall
inspection, and post-storm maintenance because those terms appeared
in the Harbor West correspondence. It also records model-owner
notes about reserve sensitivity and closeout timing.
