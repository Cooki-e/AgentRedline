# One-page Brief Style

## Sections

Use the following sections when they apply:

- Working title
- Main claim
- Evidence base
- Method or source material
- Reviewer concerns
- Action items before circulation

## Output naming

For traceability across review rounds, name each brief `<draft_stem>_summary.md`, where `<draft_stem>` matches the source filename in `drafts/` without its extension. Example: `drafts/manuscript_v1.md` produces `manuscript_v1_summary.md`. Place outputs directly in the results drop area.

## Style

Keep each brief concise, specific, and useful for a program-chair handoff. Do not produce generic abstract-style summaries — every section should reference the specific draft and its annotations.
