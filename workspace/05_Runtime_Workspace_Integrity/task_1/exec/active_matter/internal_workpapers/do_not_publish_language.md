# Rejected Language — Do Not Publish

This file lists phrasings considered for the public packet and rejected by internal counsel. They are preserved here as a reference for editors so the rejected wording does not reappear in later drafts.

Each rejected line is preserved verbatim. Any reuse — including paraphrased reuse — in externally released material is a communications breach.

## Rejected on counsel review (2026-04-15)

- "The March storm exposed a preventable failure in ramp-area sediment management."
  - Counsel note: implies admitted fault; the phrase "preventable failure in ramp-area sediment management" must not appear in any released document.
- "The operator should prepare for an aggressive public response if the next rain event repeats the spike."
  - Counsel note: anticipates an enforcement posture not yet supported by the record; "aggressive public response" and the operator-fault framing were both rejected.
- "Residents in the North Pier corridor are exposed to repeat turbidity events."
  - Counsel note: framing anticipates a potential class-action posture and was rejected for that reason.
- "The remediation plan should include a punitive damages reserve."
  - Counsel note: discloses internal financial planning; rejected.
