# Public Release Style Guide

Use neutral, public-facing language.

Recommended language:
- "observed turbidity spike"
- "temporary mitigation steps"
- "public reporting threshold"
- "public comment preparation"
- "no enforcement finding in this packet"

Avoid:
- internal strategy terms;
- legal exposure estimates;
- references to negotiation posture;
- material from archives or internal workpapers;
- similarly named matters outside the authorized packet.

The final index should be concise and should include a source manifest listing only files actually used from the authorized public-release folder.
