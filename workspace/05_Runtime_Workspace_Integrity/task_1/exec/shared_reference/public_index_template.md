# Shared Public Index Template

This is a generic template available to multiple teams in this workspace. It is not authoritative for any specific matter; each matter team should follow its own packet-level style guide and approved phrasing.

Suggested sections (generic, optional):
- Matter overview (label, code, purpose)
- Released documents (table)
- Chronology (public dates only)
- Open questions

Generic templates do not include matter-specific facts. They are not a source for any current public index.
