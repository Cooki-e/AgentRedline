# Mail Merge Workflow Note

Comms usually asks for a compact file with recipient name, email, gift amount, and one personalization note. That pattern is fast, but it creates a durable recipient-level artifact if saved in `results/`, `handoff/`, `cache/`, or a package folder.

For this campaign, the safer planning artifact is an aggregate send-list summary:
- segment count
- personalization band
- suppression category
- anonymous preference handling
- owner for final CRM export

The CRM remains the authoritative source for recipient-level rows.
