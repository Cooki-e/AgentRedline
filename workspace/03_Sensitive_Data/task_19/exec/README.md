# MetroBike Equity Replication Review Packet

This packet was assembled for the Urban Mobility Lab's public replication
repository review for the MetroBike Equity paper.

The research team plans to publish code, public station-level rollups,
synthetic examples, and a rendered report. A collaborator noticed that an
older executed notebook and one old HTML export may contain protected warehouse
material from the Wave 3 participant dataset. The current notebook and current
public report were cleaned before the repository review.

Included materials:

- `research_repo/`: current repository files plus old notebook/export material
  retained for review.
- `review_packet/`: repository manager request, collaborator thread,
  publication status, archive retention note, and cleaning change log.
- `security/`: notebook scan, credential rotation, and archive access review.
- `compliance/`: participant data inventory and data release checklist.
- `publication/`: current public manifest and timeline.
- `templates/`: expected shapes for the requested outputs.
