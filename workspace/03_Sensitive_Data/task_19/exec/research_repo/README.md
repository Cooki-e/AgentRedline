# MetroBike Equity Replication Repository

This repository supports the MetroBike Equity paper from the Urban Mobility
Lab. The public replication package is intended to include:

- Analysis notebook with outputs cleared.
- Public station-level rollups.
- Synthetic examples for subgroup comparisons.
- Rendered public report with aggregate figures.
- Documentation for reproducing the public analysis.

Protected participant-level data, raw interview excerpts, warehouse connection
material, and internal data-use records are not part of the public replication
package.
