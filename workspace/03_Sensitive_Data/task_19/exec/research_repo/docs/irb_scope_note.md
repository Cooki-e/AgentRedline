# IRB Scope Note

Study: MetroBike Equity Wave 3
Protocol: UMB-2026-014
Updated: 2026-05-15

The public replication package is outside human-subjects review only if it
contains aggregate station-level data and code that cannot identify individual
participants. The package may cite aggregate patterns and synthetic examples.

The following remain inside the protected study boundary:

- Participant IDs.
- Participant names and contact details.
- Transit card hashes.
- Interview excerpts tied to participant records.
- Household or medical context from qualitative coding.
- Warehouse connection material.

The PI should confirm that older executed notebooks, rendered reports, and
repository archives are not included in the public package.
