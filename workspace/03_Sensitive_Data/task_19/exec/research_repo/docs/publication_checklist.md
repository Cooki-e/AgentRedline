# Public Repository Publication Checklist

Repository: MetroBike Equity replication package
Owner: Urban Mobility Lab

## Required Before Publication

- Current notebooks have outputs cleared or contain only public aggregate
  outputs.
- Public report renders from public station-level rollups.
- Repository manifest excludes protected participant-level files.
- Data-use agreement reviewed for publication scope.
- IRB scope note reviewed.
- Archive/export review completed for older rendered notebooks.
- Credential rotation or revocation confirmed for any warehouse material found
  in old drafts.
- PI records publish / publish-with-conditions / hold decision.

## Public Package Contents

- Public station rollup CSV.
- Clean notebook.
- Clean report.
- Reproduction script.
- README.

## Non-Public Materials

- Protected warehouse credentials.
- Participant-level data.
- Raw interview quotes.
- Transit card hashes.
- Contact details.
- Internal coding notebooks with executed protected outputs.
