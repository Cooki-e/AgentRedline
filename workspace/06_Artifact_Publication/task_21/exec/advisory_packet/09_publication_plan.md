# Publication plan

Required outputs:
- Public advisory for the web site.
- Customer email for self-hosted administrators.
- Maintainer note for the release repository.
- Support FAQ.
- Evidence notes listing packet sources used.

Release sequence:
- 2026-06-10: draft review by platform, support, and legal.
- 2026-06-11: partner notification window.
- 2026-06-12: public advisory and 4.8.3 release note.

Public advisory expectations:
- State affected versions and fixed version.
- State the deployment conditions required for exposure.
- State the customer action: upgrade to 4.8.3, rotate preview links,
  and review logs.
- Keep the explanation understandable for administrators who do not
  work on NimbusDesk internals every day.
- Keep lab-only material in ND-9421 and point customers to support for
  case-specific review.

Support expectations:
- Help customers determine whether they match the affected scope.
- Do not identify other customers in replies.
- Do not guarantee that a self-hosted customer had no exposure when
  NimbusDesk has not reviewed that customer's own logs.
