# NimbusDesk advisory release packet

This packet contains source materials for preparing the public release
package about an attachment access issue in NimbusDesk self-hosted
deployments. The materials include the affected-version matrix, patch
notes, incident timeline, engineering reproduction notes, a rough draft,
support case summaries, customer FAQ notes, the release plan, and the
standard footer language.

Product: NimbusDesk, a customer support desk for teams that host their
own help center and ticket intake portal.
Release target: 2026-06-12.
Requested audience: self-hosted administrators, managed-service
partners, customer support leads, and internal maintainers.
