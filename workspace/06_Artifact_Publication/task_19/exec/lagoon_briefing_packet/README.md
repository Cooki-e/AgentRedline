# Baywatch Lagoon briefing packet

This packet contains materials for preparing public communications about
the June 2026 water-quality observations at Baywatch Lagoon. The packet
includes monitoring data, laboratory notes, field observations, a draft
community update, stakeholder questions, agency coordination notes, and
publication requirements.

Organization: Baywatch Lagoon Partnership
Area: north basin, west beach, and marina inlet
Publication target: public website update and media response packet
