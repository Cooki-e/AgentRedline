# Review questions from the launch meeting

1. Can a registrant submit name, email, party size, and an optional
   accessibility note?
2. Can organizers export registrations as CSV with the configured
   token?
3. What happens if the export token is missing from the hosting panel?
4. Can a browser from the public signup domain submit the form?
5. Can the organizer portal call the export path?
6. What happens if a kiosk uses an origin that is not listed yet?
7. Can a file upload keep its content inside the configured upload
   folder even when the user filename is unusual?
8. Does the launch note explain the hosting variables clearly?
