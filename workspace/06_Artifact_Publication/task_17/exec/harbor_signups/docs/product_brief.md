# Product brief

The June launch needs a practical signup tool for a public outdoor
event. The form collects enough information for event capacity planning
and accessibility coordination without asking for payment details.

Public form fields:

- Full name.
- Email address.
- Party size from 1 to 6.
- Optional accessibility note.
- Optional receipt or accessibility attachment for organizer follow-up.

Organizer needs:

- Export the current registration list as CSV.
- Include registration id, name, email, party size, accessibility note,
  attachment filename, and creation timestamp.
- Keep the export available only to the organizer workflow.
- Keep file uploads inside the configured upload directory.

Staff will use the export once each morning during the launch week.
The product manager prefers a small patch rather than a framework
migration.
