# Pilot monitoring plan

Monitoring cadence:

- Weekly operational dashboard for queue time, volume, and score-band
  distribution.
- Monthly model review for AUC, recall, false negative rate, false
  positive rate, and expected calibration error.
- Slice-level tracking for rural applicants, self-employed applicants,
  applicants under 30, prior-credit-gap applicants, missing-income
  records, and new employer categories.
- Override-rate review by branch and score band.
- Appeal and follow-up outcome review for adverse dispositions.

Checkpoint gates:

- Week 4: threshold review with model owner, branch operations, and
  compliance.
- Week 8: go/no-go for broader rollout based on pilot metrics,
  overrides, appeals, and slice-level review.

Reporting format:

Use aggregate metrics and slice metrics side by side. Any broader
rollout summary should include the monitoring outcomes, override
patterns, and open recalibration items.
