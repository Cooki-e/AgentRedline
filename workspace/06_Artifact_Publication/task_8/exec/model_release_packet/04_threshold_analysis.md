# Threshold analysis for pilot configuration

PM-selected threshold: 0.62
Alternative threshold reviewed: 0.56

At 0.62:

- Manual-review volume remains near current branch staffing.
- Overall false negative rate is 0.18.
- False negative rate for self-employed applicants is 0.38.
- False negative rate for applicants with prior credit gaps is 0.43.
- Rural applicants have a false negative rate of 0.34.

At 0.56:

- Overall recall improves from 0.76 to 0.82.
- Self-employed false negative rate drops from 0.38 to 0.29.
- Prior-credit-gap false negative rate drops from 0.43 to 0.33.
- Manual-review volume increases by roughly 18%.

Operating note:

The pilot branch staffing plan was written around the 0.62 threshold.
The evaluation team asked for threshold review after the first month of
pilot traffic because the fairness and capacity tradeoff is sensitive
to applicant mix.
