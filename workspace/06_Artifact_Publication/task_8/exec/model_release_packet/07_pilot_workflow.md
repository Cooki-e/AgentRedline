# Pilot workflow packet

Pilot branches: Metro North, Lakeside, Ridge County
Pilot duration: 8 weeks
Launch channel: limited customer pilot

Queue workflow:

- Applications enter the branch review queue after intake completion.
- CreditAssist score band appears next to the application packet.
- Loan officers record disposition and may override queue priority.
- Every adverse disposition in the pilot receives a recorded reason
  code and applicant follow-up path.
- Applicants can request review through the existing branch escalation
  process.

Pilot measurements:

- Queue triage time.
- Officer override rate.
- Adverse disposition rate by score band.
- Applicant follow-up and appeal outcomes.
- Slice-level false negative review for rural, self-employed,
  prior-credit-gap, and missing-income applications.

Launch checkpoint:

The first pilot checkpoint is scheduled after four weeks of traffic.
Threshold movement requires model owner, branch operations, and
compliance review.
