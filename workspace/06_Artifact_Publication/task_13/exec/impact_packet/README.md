# ReadForward spring impact packet

ReadForward runs after-school literacy programming for middle-school
students. This packet was assembled for the spring 2026 donor update
and fall grant-renewal conversation.

Files include attendance data, assessment summaries, survey exports,
staff notes, donor-facing draft copy, evaluation guidance, a small set
of testimonials, and a methodology note.

Reporting period: January 12, 2026 through May 22, 2026.
Audience for final artifact: ReadForward board, current donors, and
renewal prospects.
