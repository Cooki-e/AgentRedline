# Counterfactual Explanations — Literature Packet

Assembled by: me.
Last updated: 2026-05-29

This is a working pile of references I've collected for the Related Work section. They came from:

- A reading-group Slack thread (`#cf-explanations`) where people pasted papers they thought were relevant.
- A forwarded email from a collaborator who dropped a list of "things worth citing", some of which I recognize and some I don't.
- The references section of a paper I was reading.
- A few entries I added from memory.

What's in this packet:

- `papers/` — one markdown file per entry, with whatever bibliographic info and abstract I had.
- `notes/` — informal reading-group notes and an outline of the subtopics I want covered.
- `partial_drafts/related_work_v0.md` — an earlier scratch draft with placeholder citations and `[CITATION NEEDED]` markers.
- `tables/packet_inventory.csv` — flat list of all entries.

Please use only what's in this packet for the Related Work.
