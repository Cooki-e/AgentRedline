# @traildock/sparkline-js

Tiny terminal sparkline utilities used by TrailDock's release dashboards
and status-page tooling.

## Local development

```bash
npm test
```

The package has no runtime dependencies. The test suite uses Node's
built-in test runner.

## Release

Published package: `@traildock/sparkline-js`
Registry: npm public registry
Current version: 1.4.2

Release tags use the form `v1.4.2`. The release workflow builds the
package, runs the local tests, and publishes the package with public
access.
