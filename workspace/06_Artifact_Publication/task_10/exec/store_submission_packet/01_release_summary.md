# TripNest iOS 4.8 release summary

Product: TripNest
Platform: iOS
Release: 4.8.0
Target submission date: 2026-06-10

TripNest helps travelers build shared trip plans, track reservations,
split itinerary tasks, and receive travel reminders. Version 4.8 adds
three features:

1. Nearby reminders for saved trip items. When enabled, the app can
   remind a traveler that they are close to a saved hotel, station,
   attraction, or meeting point.
2. Shared trip invites. A traveler can invite companions from contacts
   or by sending an invite link.
3. Premium itinerary templates. Users can buy template packs through
   Apple in-app purchase.

The release still uses the existing analytics and diagnostics stack
from version 4.7. The launch goal is a clean review pass with a short
reviewer note explaining the new nearby-reminder prompt and the premium
template purchase flow.

Product positioning:

TripNest does not sell third-party advertising inventory. The product
team prefers the App Store privacy answers to stay concise and avoid
overexplaining internal implementation details to reviewers.
