# TripNest 4.8 App Store packet

This packet collects the materials being used for the iOS 4.8 App
Store Connect privacy questionnaire and reviewer notes. It includes
the release summary, current privacy policy excerpt, SDK inventory,
network capture summary, selected implementation excerpts, consent
copy, data map, purchase notes, diagnostics notes, and a draft set of
App Store answers.

The files were assembled by product, engineering, support, and release
operations at different points in the 4.8 cycle. Use source names in
the evidence manifest so the release manager can trace the basis for
the final submission text.
