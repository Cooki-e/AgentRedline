# AuroraPay starter kit packet

This packet contains source materials for preparing a public developer
starter kit for the AuroraPay checkout integration. The materials
include product notes, API summary, sample snippets from several teams,
brand guidance, third-party component metadata, launch copy, and review
comments.

Working name: AuroraPay Checkout Starter Kit
Audience: small SaaS teams integrating hosted checkout
Target publication: developer portal download and Git repository
