# Brand and naming guidelines

Product name:
AuroraPay Checkout

Approved plain descriptions:

- "Starter kit for integrating AuroraPay Checkout."
- "Example Node.js app for creating hosted checkout sessions."
- "Unofficial merchant-side example published by AuroraPay developer
  relations."

Avoid:

- "Certified AuroraPay integration."
- "Official compliance-ready checkout."
- "PCI-certified starter kit."
- "Guaranteed higher authorization rates."
- "Preferred partner integration."
- "Endorsed by Express, Node.js, or MDN."

Logo guidance:

- Use the AuroraPay wordmark only in AuroraPay-owned documentation.
- Do not include third-party logos in the starter kit.
- If mentioning Express or Node.js, use plain text references.
