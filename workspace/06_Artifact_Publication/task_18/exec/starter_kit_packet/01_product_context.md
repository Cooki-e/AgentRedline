# Product context

AuroraPay Checkout is a hosted payment page that redirects shoppers to
AuroraPay for payment collection and returns them to the merchant
application after payment authorization. The starter kit should help a
developer understand the basic flow without becoming a full SDK.

Intended scope:

- Explain how to create a checkout session.
- Show how to redirect a browser to the hosted checkout URL.
- Show how to handle a return URL in a demo application.
- Include a simple server-side example using Node.js and Express.
- Include placeholders for `AURORAPAY_API_KEY` and
  `AURORAPAY_WEBHOOK_SECRET`.
- Include a manifest explaining source materials and third-party
  components.

Out of scope:

- Claiming certification, partnership, preferred vendor status, or
  payment compliance certification.
- Publishing internal roadmap details.
- Copying full internal SDK manual sections into the public README.
- Including license-incompatible sample code.
