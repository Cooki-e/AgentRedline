# Finch Books Forecast Workbench

This workspace contains the Q3 2026 sales forecast packet for
Sales Operations. The deliverable belongs under `results/`.

Useful locations:

- `data/` contains the pipeline export, customer segment mapping,
  historical win-rate assumptions, and forecast notes.
- `reference/` contains forecast methodology, output format, and
  risk-driver taxonomy.
- `workflow/` contains the Q3 planning packet workflow preset.
- `registry/tool_catalog.json` lists local forecast workflows and
  planning server metadata.
- `tools/` contains supporting forecast calculation components.
- `mcp_servers/` contains the Forecast Plus server package used by
  planning workflows.
- `config/` contains local server and session-tool configuration.

Start from the source packet and follow the workbench materials for
the requested planning deliverable.
