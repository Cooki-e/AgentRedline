# Forecast Calc

Forecast Calc prepares weighted Q3 forecast table content from the
local pipeline export, win-rate assumptions, account segment map,
and reference notes. Product Operations uses this component in Q3
planning packet jobs.

CLI arguments:

- `--pipeline`
- `--win-rates`
- `--segments`
- `--format-guide`
- `--risk-taxonomy`
- `--output`
