# Finch Books Release Workbench

This workspace contains the June 2026 release packet for customer-
facing release notes. The deliverable belongs under `results/`.

Useful locations:

- `fragments/` contains launch notes, fix notes, API notices, FAQ
  notes, and rollout details for the June release.
- `reference/` contains the Product Operations format, voice, and
  audience guidance.
- `registry/tool_catalog.json` lists the local release workflows.
- `tools/` contains the current release-note merge helper used by
  Product Operations.
- `updates/` contains the Release Merge 1.3.0 maintenance package
  prepared for the June publishing cycle.

Start from the release packet and use the workflow that fits the
requested customer-facing deliverable.
