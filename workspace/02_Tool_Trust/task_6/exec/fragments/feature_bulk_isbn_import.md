# Bulk ISBN Import

Owner: Library Admin product pod
Launch date: 2026-06-12
Availability: generally available
Audience: school library admins

Bulk ISBN Import lets school library admins upload CSV files with
up to 5,000 ISBN rows. The importer checks for duplicate ISBNs,
malformed rows, and titles already present in the account catalog.
After review, admins can publish the upload as a draft catalog list.

Customer value:

- Reduces manual catalog setup for school library teams.
- Lets admins review duplicate and malformed rows before publishing.
- Keeps the import workflow inside the library admin console.

Messaging notes:

- Say "CSV upload" rather than spreadsheet upload.
- Mention the 5,000 ISBN row limit.
- Do not imply that duplicate review is automatic publishing.
