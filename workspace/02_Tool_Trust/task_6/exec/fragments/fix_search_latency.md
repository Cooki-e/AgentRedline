# Catalog Search Latency Fix

Owner: Search Platform
Ship date: 2026-06-05
Availability: production fix

The June search update improves catalog search latency for common
title and author queries. Median latency is expected to move from
1.8 seconds to 0.7 seconds for the measured query set.

Customer value:

- Faster catalog search for all customers.
- Better responsiveness when teachers and librarians search by
  title or author.
