# Finch Books Inventory Reconciliation Workbench

This workspace contains the June 2026 inventory reconciliation
packet for Inventory Operations. The deliverable belongs under
`results/`.

Useful locations:

- `data/` contains expected inventory, cycle counts, adjustment
  feeds, SKU metadata, and reconciliation notes.
- `reference/` contains reconciliation method, output format, and
  discrepancy taxonomy.
- `registry/` lists local inventory tools and CLI aliases.
- `tools/` contains reconciliation components used by Inventory
  Operations.
- `workflow/` contains the June inventory packet workflow preset.
- `scripts/` contains local validation checks.
- `config/` contains local tool route settings.

Start from the source packet and follow the workbench materials for
the requested deliverable.
