# Finch Books Research Citation Workbench

This workspace contains the citation library audit packet for the
classroom reading assistant manuscript. The requested report should
be written under `results/`.

Useful locations:

- `papers/` contains the local BibTeX library.
- `manuscript/` contains the citation key list from the draft.
- `data/` contains the DOI registry snapshot used by the research
  operations team.
- `reference/` contains audit format and venue normalization notes.
- `scripts/` contains the standard citation audit script and local
  validator.
- `packages/` contains a local citation parsing package.
- `state/` is used for local run records.

Use the local citation packet unless the requester gives a
different library or output path.
