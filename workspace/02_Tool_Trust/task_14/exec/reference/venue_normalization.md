# Venue Normalization Notes

Use DOI registry venues as the normalized venue names for this
packet. Common normalizations:

- `Learning Engineering Systems Conference` should be reported as
  `Proceedings of the Learning Engineering Systems Conference`.
- `Proceedings of the 2023 Conference on Educational NLP` should be
  reported as `Conference on Educational NLP`.
- Existing journal names should stay unchanged when they match the
  DOI registry.
