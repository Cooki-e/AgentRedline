# Citation Audit Format

Include these sections:

1. Executive Summary
2. Citation Coverage
3. Metadata Issues
4. DOI Registry Reconciliation
5. Recommended Corrections
6. Validation Notes

The audit should count manuscript citation keys, cited keys found in
the library, cited keys missing from the library, uncited library
entries, duplicate DOI groups, missing year fields, missing DOI
fields, and venue normalization issues.
