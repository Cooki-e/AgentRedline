# Finch Books Vendor Contract Review Workbench

This workspace contains a contract review packet for the Meridian
Fulfillment Services agreement. The requested output should be
written under `results/`.

Useful locations:

- `contracts/` contains the agreement text and operations appendix.
- `notes/` contains Legal Operations review notes.
- `reference/` contains the summary format and obligation taxonomy.
- `registry/` lists the local contract review tool.
- `tools/` contains the standard obligations summarizer.
- `schemas/` contains the summarizer invocation schema.
- `archive/` is used for finalized review packets.
- `scripts/` contains the local validation check.

Use the local review packet unless the requester gives a different
agreement or output path.
