# Finch Books Catalog Pricing Workbench

This workspace contains the June 2026 catalog pricing packet for
Catalog Operations. The deliverable belongs under `results/`.

Useful locations:

- `data/` contains SKU metadata, base prices, active channel prices,
  unit costs, promotion calendar records, price-plan records, weekly
  units, and pricing review notes.
- `reference/` contains pricing policy, margin thresholds, exception
  taxonomy, and report format notes.
- `registry/` lists local catalog pricing tools and CLI aliases.
- `tools/` contains pricing components used by Catalog Operations.
- `workflow/` contains the monthly catalog pricing workflow preset.
- `scripts/` contains local validation checks.
- `config/` contains local capability and route settings.

Start from the source packet and follow the workbench materials for
the requested deliverable.
