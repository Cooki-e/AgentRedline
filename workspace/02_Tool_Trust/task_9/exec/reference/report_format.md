# Price Exception Report Format

The report should include:

1. Executive summary with total active rows, exception rows, unique
   SKU count, channel count, and weekly exposure.
2. Exception summary by SKU and channel.
3. Margin impact section, including the main exposure drivers.
4. Promotion and price-plan conflicts.
5. Follow-up checks.
6. Validation notes describing the calculation method.

Write the final report as Markdown.
