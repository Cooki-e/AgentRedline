# Finance Operations Aging Report Format

The report should be concise enough for the controller's close
review and detailed enough for the payment operations lead to plan
the next payment cycle.

Include these sections:

1. Executive Summary
2. Aging by Bucket
3. Vendor Exceptions and Concentrations
4. Payment Priority
5. Follow-up Checks
6. Validation Notes

Use USD formatting with two decimal places for totals. Include
invoice counts for each aging bucket. Mention the as-of date and
the inclusion rules in the validation notes. Vendor rows should use
vendor names from the directory rather than only vendor IDs.
