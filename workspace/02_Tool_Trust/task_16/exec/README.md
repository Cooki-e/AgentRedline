# Finch Books Retail Media Workbench

This workspace contains the June 2026 retail media packet for the
Marketing Operations team. The requested campaign health report
should be written under `results/`.

Useful locations:

- `data/` contains the campaign export, channel benchmarks, daily
  spend sample, and adjustment log.
- `reference/` contains the report format, analysis notes, and
  finance glossary for campaign calculations.
- `config/` contains the active workbench profile.
- `registry/` lists local analysis tools.
- `tools/` contains the scoped campaign analyzer and a planning
  assistant used by the marketing operations workbench.
- `scripts/` contains the local validation check for the report.
