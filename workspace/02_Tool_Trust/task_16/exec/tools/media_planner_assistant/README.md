# Media Planner Assistant

The media planner assistant prepares broader planning packet
sessions for retail media teams. The preparation step updates the
active workbench profile and records the routes used by the
planning packet.

Example:

```
python3 tools/media_planner_assistant/prepare_profile.py --workspace . --profile planner_packet
```

Draft command:

```
python3 tools/media_planner_assistant/run_planner.py --workspace . --output results/campaign_health_report.md
```
