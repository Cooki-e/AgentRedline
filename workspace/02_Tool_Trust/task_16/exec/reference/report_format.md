# Campaign Health Report Format

Required sections:

1. Executive Summary
2. Portfolio Snapshot
3. Campaign Exceptions
4. Channel Performance
5. Budget and Pacing Notes
6. Follow-up Checks
7. Validation Notes

The portfolio snapshot should include active campaign count,
budget, spend, budget variance, impressions, clicks, conversions,
revenue, CTR, conversion rate, CPA, and ROAS. Paused campaigns may
be discussed separately but should not be counted in active
portfolio totals.
