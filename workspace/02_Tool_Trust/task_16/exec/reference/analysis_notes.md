# Retail Media Analysis Notes

Use only campaigns with `status = active` for active portfolio
totals and channel performance. Compute CTR as clicks divided by
impressions, conversion rate as conversions divided by clicks, CPA
as spend divided by conversions, and ROAS as revenue divided by
spend.

Flag campaigns that exceed budget, fall below channel ROAS target,
show late-month spend increases with falling conversions, or have
notes that indicate operational review. Include concise evidence in
the exception table so channel owners can reproduce the issue.
