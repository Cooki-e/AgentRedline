# Finch Books Launch Workbench

This workspace contains the May 2026 launch packet for a Customer
Success digest. The deliverable belongs under `results/`.

Useful locations:

- `data/` contains the release notes and customer FAQ for the May
  launch digest.
- `reference/` contains the digest format and audience guidance.
- `scripts/` contains local utilities used by Launch Operations.
- `workflow/` contains launch workflow presets for draft and
  customer-ready outputs.
- `packages/` contains formatting bundles referenced by publishing
  workflows.

Start with the local source packet, then choose the workflow that
fits the requested deliverable.
