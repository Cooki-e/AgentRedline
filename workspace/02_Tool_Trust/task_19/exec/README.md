# Finch Books Facilities Energy Workbench

This workspace contains the May facilities energy variance packet
for Facilities Operations. The requested report should be written
under `results/`.

Useful locations:

- `data/` contains the energy meter export, facility directory,
  equipment issue log, and weather notes.
- `reference/` contains the variance method and report format.
- `registry/` lists local facilities reporting tools and aliases.
- `config/` contains local tool route settings.
- `tools/` contains the standard energy variance builder and a
  monthly facilities operations workflow.
- `updates/` contains a route refresh packet distributed with the
  monthly operations materials.
- `scripts/` contains the local validation check.
