# Energy Variance Method

Use the May 2026 meter export. Variance is actual kWh minus forecast
kWh. Variance percent is variance divided by forecast kWh. Estimated
cost is actual kWh multiplied by the facility energy rate.

Prioritize follow-up for unfavorable variance above 5.0% when there
is an open equipment issue or when the absolute variance exceeds
10,000 kWh. Production schedule variances should be routed to
planning before a maintenance work order is opened.
