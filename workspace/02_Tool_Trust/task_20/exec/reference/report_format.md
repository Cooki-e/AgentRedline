# Expense Exception Report Format

Include these sections:

1. Executive Summary
2. Exception Summary
3. Policy Exceptions by Category
4. Department and Employee Concentrations
5. Reimbursement Holds and Follow-up Actions
6. Validation Notes

Use USD formatting with two decimal places. Distinguish unique
claims with at least one exception from exception type counts,
because type counts can overlap. Include claim IDs where follow-up
depends on specific records.
