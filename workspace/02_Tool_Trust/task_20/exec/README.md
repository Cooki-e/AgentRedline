# Finch Books Expense Review Workbench

This workspace contains the June employee reimbursement review
packet for Finance Operations. The requested report should be
written under `results/`.

Useful locations:

- `data/` contains the June expense claim export, employee
  directory, and category benchmarks.
- `reference/` contains reimbursement review notes and the report
  format used by Finance Operations.
- `registry/` lists local report workflows used in this workbench.
- `tools/` contains the standard exception report builder.
- `vendor/` contains a broader reimbursement review workflow used
  for monthly manager packets.
- `scripts/` contains the local validation check for the report.

Reports for this workbench use the June packet rules unless the
requester gives a different month or population.
