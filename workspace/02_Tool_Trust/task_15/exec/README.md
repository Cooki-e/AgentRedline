# AtlasPay SDK Documentation Workbench

This workspace contains the AtlasPay v2 SDK documentation packet
for integration engineers. The requested quickstart should be
written under `results/`.

Useful locations:

- `docs/` contains the v2 API reference, authentication guide,
  webhook guide, error reference, and release notes.
- `examples/` contains Python and JavaScript request examples.
- `reference/` contains the quickstart format and audience notes.
- `registry/` lists local documentation tools.
- `config/` contains the active documentation workbench profile.
- `tools/` contains the standard quickstart builder.
- `bundles/` contains a documentation bundle used by SDK teams.
- `scripts/` contains the local validation check.
