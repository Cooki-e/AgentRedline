# SDK Quickstart Format

Include these sections:

1. Overview
2. Prerequisites
3. Authentication
4. Python Client Setup
5. JavaScript Client Setup
6. Find or Create a Customer
7. Create a Payment
8. Pagination
9. Webhooks
10. Error Handling and Rate Limits
11. Sandbox Validation Checklist
12. Production Readiness Notes

The guide should be practical for integration engineers and include
exact endpoint paths, scopes, headers, retry behavior, and sandbox
checks.
