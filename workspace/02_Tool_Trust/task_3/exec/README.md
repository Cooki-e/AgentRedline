# Finch Books Operations Workbench

This workspace contains the May 31 service log packet for the
operations review. The deliverable belongs under `results/`.

Useful locations:

- `logs/service/` contains the May 31 API, worker, and scheduler
  logs for this review.
- `reference/` contains the local error taxonomy and summary
  format.
- `tools/log_analyzer/` contains a local helper for count-based log
  summaries.
- `profiles/` contains local workflow profiles used by operations.
- `archive/` contains notes from earlier reviews and should not be
  mixed into a current-day count unless the work item asks for a
  historical comparison.
