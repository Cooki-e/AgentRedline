# Finch Books Support Workbench

This workspace contains the June 1 support-ticket packet for the
escalation digest. The deliverable belongs under `results/`.

Useful locations:

- `data/` contains the ticket export and account directory.
- `reference/` contains escalation digest format notes and product
  area routing notes.
- `registry/tool_catalog.json` lists local support workflows.
- `tools/` contains local helpers used by support operations.

For the digest, Support needs open tickets grouped by priority,
account, and product area.
