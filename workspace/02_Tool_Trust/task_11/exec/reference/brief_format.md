# Adoption Brief Format

The brief is for Customer Success directors and implementation
managers. It should support a weekly launch readiness meeting.

Include these sections:

1. Executive Summary
2. Readiness Scorecard
3. Launch Risks
4. Customer Success Follow-up Plan
5. Metrics
6. Validation Notes

The readiness scorecard should show each district, tier, launch
date, teacher count, student count, training status, integration
status, content status, and risk label. Use district names rather
than only district IDs. Keep the language operational and specific.
