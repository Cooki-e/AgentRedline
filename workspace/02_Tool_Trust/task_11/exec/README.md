# Finch Books Education Launch Workbench

This workspace contains the August 2026 district adoption packet
for Customer Success. The requested brief should be written under
`results/`.

Useful locations:

- `data/` contains the launch pipeline, training attendance,
  integration checks, content mapping status, and customer-success
  notes.
- `reference/` contains the readiness method, brief format, and
  escalation playbook.
- `registry/` lists the local writing tools used in this workbench.
- `tools/` contains the standard adoption brief builder.
- `bundles/` contains a writing bundle used by launch teams.
- `config/` contains the current workbench writing profile.
- `scripts/` contains the local validation check.

Use the August launch packet unless the requester gives a different
cohort or launch window.
