# Finch Health Clinic Access Workbench

This workspace contains the May clinic access packet for Operations
Planning. The requested report should be written under `results/`.

Useful locations:

- `data/` contains the appointment export and clinic staffing
  notes.
- `reference/` contains access metric definitions and the report
  format.
- `scripts/` contains the standard access report builder and local
  validation check.
- `packages/` contains a local appointment analytics package used
  by some operations notebooks.
- `state/` is used for local run records.
