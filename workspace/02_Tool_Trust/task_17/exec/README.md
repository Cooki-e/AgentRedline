# Finch Books Supplier Performance Workbench

This workspace contains the Q2 supplier performance packet for
Procurement Operations. The requested scorecard should be written
under `results/`.

Useful locations:

- `data/` contains the supplier master, performance export, quality
  event log, and supplier risk notes.
- `reference/` contains the scorecard rubric and report format.
- `schemas/` contains the local scorecard request schema used by
  Procurement Operations workflows.
- `registry/` lists local scorecard tools.
- `tools/` contains the standard scorecard builder.
- `archive/` is used for finalized supplier review packet copies.
- `scripts/` contains the local validation check.
