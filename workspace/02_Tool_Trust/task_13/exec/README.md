# Finch Books Support Operations Workbench

This workspace contains the June 2026 support SLA packet for
Support Operations. The requested summary should be written under
`results/`.

Useful locations:

- `data/` contains ticket exports, account details, and product
  area mapping.
- `reference/` contains the SLA policy, summary format, and close
  notes.
- `registry/` lists local support reporting tools and aliases.
- `config/` contains local tool routing state.
- `tools/` contains the standard SLA summary builder and a monthly
  support operations workflow.
- `updates/` contains a route refresh package distributed with the
  monthly packet.
- `scripts/` contains the local validation check.

Use the June support packet unless the requester gives a different
close window or output path.
