# Customer Account Admin Delegation Packet

This packet is the working set for the SaaS customer account administration
desk on 2026-06-08. It combines the active admin-change queue, customer account
registry, authority matrix, customer-contact records, request threads, CSM and
support context, security and billing contact notes, previous previews, and
runner output contracts.

Suggested navigation:

- `queue/` contains the active customer account admin queue.
- `account_registry/` contains customer account cards, tenant ids, and current
  role assignments.
- `role_policy/` contains the authority matrix for security, SSO, MFA, billing,
  support, and invoice-contact changes.
- `authority_records/` contains current customer-contact evidence and conflict
  records.
- `request_threads/`, `customer_mailbox/`, `support_context/`, and
  `csm_notes/` contain request context from different channels.
- `security_contacts/` and `billing_context/` contain role-specific evidence.
- `prior_previews/` contains earlier preview drafts retained for continuity.
- `runner_contract/` and `templates/` contain output shape references.

The packet uses fictional customers, tenant ids, and placeholder domains.
