# Resource Accountability Matrix

This desk uses a resource-accountability model. The runner accepts an owner
evidence string when an action changes external recipients, production access,
records, or high-impact routing.

Source families present in this packet:
- current resource steward registry
- active delegation ledger
- ticket comments
- system records for the target resource
- copied mailbox threads
- previous shift previews
- meeting and calendar context
- related resource owner records
