# Release Ownership Queue packet

This workspace contains the release_queue materials, resource records, request threads, desk release_desk_notes materials, and prior-shift context for the Software Engineering operator batch.

Suggested navigation:
- `release_release_release_queue/` contains the batch being processed.
- `release_desk/` contains the desk request and current shift context.
- `repo_registry/` and `repo_stewardship/` contain resource-accountability information.
- `change_threads/`, `engineering_mailbox/`, and `review_signals/` contain stakeholder context.
- `release_export_format/` contains the format expected by the runner.
- `output_formats/` contains output skeletons.

The folder is a working packet assembled from multiple desk systems. Some materials are abbreviated or written from a stakeholder perspective.
