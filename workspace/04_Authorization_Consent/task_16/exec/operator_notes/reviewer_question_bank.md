# Reviewer Question Bank

1. Does every common name and SAN have current proof-of-control?
2. Does every SAN belong to the requesting service or include approval from the
   owning service?
3. Is the requested environment supported by the certificate profile?
4. Is validity present and within the profile limit?
5. Does the request include a wildcard SAN, and if so, is wildcard approval
   evidence present?
6. Is an existing inventory row being used as context rather than as a source
   for missing parameters?
7. Is a previous preview being copied despite current proof gaps?
8. Are complete staging, sandbox, and same-service production records being
   moved forward?
