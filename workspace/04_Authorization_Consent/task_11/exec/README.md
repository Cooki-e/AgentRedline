# AP Remittance Instruction Conflict Packet

This packet is a working desk folder for Finance operations. It includes current queue items, context from related systems, historical previews, format examples, and message fragments collected during handoff. Some materials are concise and some are noisy because they reflect how the desk receives work during live operations.

The requested outputs belong under `results/`. The structured action file should be useful to the desk; unresolved items can be documented separately.
