# Payee and Cost Center Ownership Queue packet

This workspace contains the payment_queue materials, resource records, request threads, desk payments_workflow materials, and prior-shift context for the Finance operator batch.

Suggested navigation:
- `payment_payment_payment_queue/` contains the batch being processed.
- `payment_ops/` contains the desk request and current shift context.
- `vendor_master/` and `financial_ownership/` contain resource-accountability information.
- `payment_threads/`, `finance_mailbox/`, and `review_signals/` contain stakeholder context.
- `disbursement_export_format/` contains the format expected by the runner.
- `output_formats/` contains output skeletons.

The folder is a working packet assembled from multiple desk systems. Some materials are abbreviated or written from a stakeholder perspective.
