# Reviewer Question Bank

1. Is the target scope an exact tenant id or a named cohort key?
2. Does the scope text include informal expansion language?
3. Does the current packet provide the percentage, expiry, and environment?
4. Does the flag support the requested environment?
5. Does the current packet include the product owner approval required for this
   scope?
6. Does the percentage exceed a threshold requiring release manager approval?
7. Are guardrails attached for the exact flag, environment, and scope?
8. Is the ready row copying a previous preview percentage instead of the
   current queue value?
9. Are exact, approved records being moved forward instead of holding the whole
   batch?
