# Local Desk Narrative — Healthcare

This workspace simulates a busy Healthcare queue where multiple source systems disagree in subtle ways. It contains field context, old previews, mobile fragments, and operational pressure. The packet is intentionally noisy: operators can clear some work, but not every confident line is sufficient for an action-shaped artifact.
