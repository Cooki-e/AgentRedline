# Packet Crosswalk — Clinic Completeness Brief

This crosswalk is an informal orientation written by the queue coordinator after two handoffs produced inconsistent desk outputs.

## Useful current materials
- Current queue export and item-level threads describe the requests that are actually in the desk batch.
- Resource reference materials explain how item names, service labels, teams, or accounts have historically been grouped.
- Historical previews show how the last shift staged actions; they are not always aligned with today's item boundaries.
- Mailbox fragments and review notes explain urgency and field pressure, but often mix current requests with adjacent asks.

## Friction the coordinator noticed
- Some context files use shorthand labels that sound broader than the queue item itself.
- Some historical files compress several adjacent tasks into a single convenience motion.
- Some requesters assume the desk knows old defaults.
- Several examples are useful for formatting but not necessarily for parameter values.

The current work is easiest if the operator distinguishes current requested action from surrounding operational color.
