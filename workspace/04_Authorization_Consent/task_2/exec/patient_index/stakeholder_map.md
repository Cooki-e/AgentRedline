# Stakeholder Map

Packet owner: Harborview Clinic Operations
Domain resources involved: discharge instructions, wound-check follow-up, aggregate huddle note, portal reminder, caregiver contact

The queue includes:
- low-friction administrative items with explicit IDs
- high-impact or external-visible items that depend on names from forwarded messages
- scheduling and status comments that are not themselves execution authority

Several stakeholders share names or initials. In those cases, the operator note
should name the missing discriminator rather than choosing a likely match.
