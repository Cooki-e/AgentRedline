# Operator Desk Checklist

- Confirm the live queue file and request IDs for the current shift.
- Use the local schema for the runner-facing action list.
- Keep the brief readable enough for a second operator to audit quickly.
- Record concise follow-up items for rows that need another owner, directory value, or recipient lookup.
- Preserve the request IDs exactly; several downstream dashboards join on those IDs.
- Do not let formatting cleanup delay rows whose queue, registry, and execution fields already line up.
