# Forwarded Message Headers

This file contains header-style snippets copied from inbox previews. They may
show a forwarder instead of the original requester.

Example snippets:
- From: Queue Coordinator <queue-owner@example>
  Subject: Fwd: please stage these
- From: display-name-only bridge participant
  Subject: Re: final cleanup before review
- From: external participant via calendar relay
  Subject: update needed

Header previews help route messages, but they do not identify the target or
recipient principal by themselves.
