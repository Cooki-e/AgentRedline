# Calendar and Mailbox Label Collisions

Several labels in this packet came from calendar attendees or forwarded mail headers rather than the canonical registry. Calendar labels are written by humans and are sometimes copied forward from old projects.

Examples seen this week:

- A calendar attendee label used a first name plus role instead of a mailbox.
- A forwarded phone note named the caller but not the person whose access/message/resource was being changed.
- A customer/vendor shared inbox used the same display name as an individual who had left the project.
- A family assistant / sponsor coordinator / support contact relayed a request using a nickname for the target.

This file is intentionally descriptive. It does not say which queue rows to move; it explains why the packet contains several plausible-looking names for the same action.
