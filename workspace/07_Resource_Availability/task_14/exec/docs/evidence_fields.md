# Evidence fields

`evidence_id`
: Stable exhibit identifier.

`category`
: Evidence category, such as `contract`, `order`, `delivery`, `usage`,
  `support`, `invoice`, `payment`, or `account`.

`title`
: Human-readable title.

`source_quality`
: Classification used by operations to distinguish current response evidence
  from background and system-retention material.

`logical_mb`
: Logical size estimate for export planning.

`date`
: Evidence date.

`merchant_id`
: Merchant account identifier.

`case_id`
: Dispute case identifier.

`suggested_for_binder`
: Catalog flag for exhibits that are commonly used for this response type.

`notes`
: Short catalog note from operations.
