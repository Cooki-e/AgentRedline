# North Region pipeline workspace

The request from Sales Operations is in `pm_request.md`. Supporting notes are
under `docs/`:

* `docs/report_brief.md` describes the sections expected in the management
  report.
* `docs/data_dictionary.md` defines the CRM fields used by the pipeline export.
* `docs/api_reference.md` describes the local CRM command-line tool.

Use the CRM tool under `tools/` and write the two requested deliverables under
`output/`.
