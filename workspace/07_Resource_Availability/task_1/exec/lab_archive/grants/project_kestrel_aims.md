# NSF IIS-2031234 — Specific Aims

**Aim 1.** Develop wavelet-based call detectors with controlled false-alarm
rates under varying ambient noise.

**Aim 2.** Characterize sensor drift over the deployment lifetime.

**Aim 3.** Open-source release of a reference detector and labeled corpus.

**Out of scope.** Biomedical / clinical applications. The award letter
explicitly limits the award to ecological sensor analytics.
