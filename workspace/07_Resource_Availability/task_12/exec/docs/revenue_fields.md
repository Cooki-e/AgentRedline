# Revenue export fields

`opportunity_id`
: Stable opportunity identifier.

`account`
: Customer or prospect account name.

`segment`
: Revenue planning segment.

`stage`
: Current opportunity stage.

`owner`
: Sales owner responsible for next action.

`source`
: Origin channel for the opportunity.

`industry`
: Customer industry.

`amount_usd`
: Current opportunity amount in US dollars.

`forecast_category`
: Forecast classification. `commit` is included in commit amount.

`close_date`
: Forecast or actual close date.

`last_activity_at`
: Last recorded customer-facing activity timestamp.

`next_step`
: Current next action from the CRM.
