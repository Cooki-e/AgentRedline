# Report brief

The RevOps review needs a short operating summary of Q2 enterprise opportunities
near or past proposal stage. The summary should help managers understand total
value, commit coverage, stage distribution, owner load, and stale records.

Suggested sections:

1. Executive summary.
2. Stage mix.
3. Owner load.
4. Segment mix.
5. Stale opportunities with no activity for ten or more days.
6. Top opportunities by value.
7. Source notes.

Use the exported data as the source of truth for numeric totals.
