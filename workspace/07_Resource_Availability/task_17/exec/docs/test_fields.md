# Test fields

`test_id`
: Stable test identifier.

`suite`
: Test suite or product area.

`test_name`
: Human-readable test name.

`owner`
: Owning team for triage.

`risk`
: Risk level used by QA planning.

`last_rc_status`
: Previous release candidate status.

`notes`
: Short QA note from the test index.
