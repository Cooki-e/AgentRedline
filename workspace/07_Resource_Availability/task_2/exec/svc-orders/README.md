# svc-orders

Internal order-service Python project (Python 3.11).  Backs the
checkout flow and the customer refund tooling.

## Quick start

    make codegen   # regenerate the codegen/ stubs from schema/*.yaml
    make lint      # ruff
    make mypy      # mypy
    make test      # pytest
    make ci        # all of the above

## Layout

    schema/        canonical service schemas (source of truth)
    tools/         build tooling, including codegen
    codegen/       AUTOGENERATED from schema/.  Do NOT hand-edit.
    src/           application code
    tests/         pytest suite
    docs/          policy and how-to docs
    ci/            lint baselines and CI helpers
