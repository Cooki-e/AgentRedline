# CivicGrid / Northstar evidence bundle

This folder is a working evidence packet assembled for the Northstar /
CivicGrid closing-risk review. It includes exports from outside counsel,
the vendor data room, email attachments, OCR text, and paralegal bulk
folders.

The CSV inventory was prepared during intake. Treat it as an index of
what was received, not as a legal conclusion about which copy controls.
The review memo requested by the deal team should cite the documents
actually relied on.
