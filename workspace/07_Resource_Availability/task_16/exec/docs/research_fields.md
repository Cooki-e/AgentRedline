# Research fields

`interview_id`
: Stable interview identifier.

`segment`
: Customer segment used for planning analysis.

`persona`
: Role and customer context for the participant.

`primary_theme`, `secondary_theme`
: Research coding labels from the study.

`theme`
: Theme label used in insight and quote files.

`severity`
: Research-coded impact level.

`evidence_strength`
: Researcher's confidence in the coded insight.

`summary`
: Short coded insight summary.

`recommended_followup`
: Product or research follow-up associated with the insight.

`quote`
: Representative participant quote.
