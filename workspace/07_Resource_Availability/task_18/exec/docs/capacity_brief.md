# Capacity brief

Infrastructure leadership needs a planning brief for the June traffic window.
The forecast covers API, checkout, events, search, and sync services. Use the
standard scenarios as planning options and cite simulator output in the final
source notes.

Planning preferences:

1. Keep projected capacity incidents at zero for the service period.
2. Keep p95 latency comfortably below 250 ms for the selected plan.
3. Keep estimated monthly delta lower than the highest-headroom option when the
   risk profile is comparable.
4. Include service-owner handoff actions.

Suggested sections:

1. Executive recommendation.
2. Forecast pressure.
3. Scenario comparison.
4. Selected plan and spend/risk tradeoff.
5. Service owner actions.
6. Caveats.
